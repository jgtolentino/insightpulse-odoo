Slack Bridge
=

Bidirectional Slack integration - OAuth, Events API, sync

Installation
------------

Install this module via Odoo Apps or CLI:

.. code-block:: bash

    docker-compose exec odoo /opt/odoo/odoo-bin \
      -c /etc/odoo/odoo.conf -d odoo19 \
      -i ipai_slack_bridge --stop-after-init
