Installation
============

This module requires the following OCA dependencies:

* queue_job
* report_xlsx
* server_environment

To install this module, you need to:

1. Install the required OCA modules
2. Install this module through the Odoo Apps interface
3. Configure expense policies and approval workflows

Configuration
=============

To configure this module, you need to:

1. Go to Expense → Configuration → Settings
2. Define expense policies and limits
3. Configure approval workflows
4. Set up OCR audit settings

Usage
=====

Expense Management Workflow
---------------------------

1. **Cash Advance Requests**
   - Employees request cash advances for business expenses
   - Multi-level approval process
   - Integration with accounting for disbursement

2. **Expense Submission**
   - Submit expenses with receipt attachments
   - Automatic policy validation
   - OCR-based receipt analysis

3. **Policy Enforcement**
   - Automatic policy compliance checking
   - Exception handling and escalation
   - Audit trail for all expense transactions

4. **OCR Audit Process**
   - Automated receipt scanning and data extraction
   - Fraud detection and anomaly identification
   - Audit report generation

Key Features
------------

* **Cash Advance Management**: Complete lifecycle from request to reconciliation
* **Expense Policy Engine**: Configurable policies with automatic enforcement
* **OCR Integration**: Automated receipt processing and data extraction
* **Audit Trail**: Comprehensive tracking of all expense activities
* **Reporting**: Advanced analytics and compliance reporting
