Pulser Hub Sync

GitHub App integration for Odoo 19, providing webhook listeners and OAuth authentication for the pulser-hub GitHub App.

üîê OAuth 2.0 Authentication

Seamless GitHub App installation with OAuth callback handling at /odoo/github/auth/callback

üé£ Webhook Listener

Configuration

Set the following environment variables:

export GITHUB_APP_ID=2191216
export GITHUB_APP_CLIENT_ID=Iv23liwGL7fnYySPPAjS
export GITHUB_APP_PEM_PATH=~/.github/apps/pulser-hub.pem
export GITHUB_APP_CLIENT_SECRET=your_client_secret_here

Usage

  1. Install the module via Apps menu
  2. Navigate to GitHub ‚Üí Integrations
  3. Install the pulser-hub app on your GitHub account
  4. Configure webhook secret for signature verification
  5. Monitor webhook events in GitHub ‚Üí Webhook Events

Webhook Configuration

Configure your GitHub App webhook to point to:

https://your-odoo-domain.com/odoo/github/webhook

Select the events you want to receive:

  • Push events
  • Pull request events
  • Issue events
  • Workflow run events

Technical Details

  • Client ID: Iv23liwGL7fnYySPPAjS
