Installation
============

This module requires the following OCA dependencies:

* queue_job
* base_tier_validation
* report_xlsx
* server_environment

To install this module, you need to:

1. Install the required OCA modules
2. Install this module through the Odoo Apps interface
3. Configure procurement settings as needed

Configuration
=============

To configure this module, you need to:

1. Go to Procurement → Configuration → Settings
2. Configure approval workflows and tiers
3. Set up vendor catalogs and scoring criteria
4. Configure RFQ round settings

Usage
=====

Procurement Workflow
--------------------

1. **Purchase Requisition Creation**
   - Create purchase requisitions with line items
   - Submit for approval through tier validation

2. **RFQ Rounds**
   - Automatic RFQ generation from approved requisitions
   - Multi-round bidding process
   - Vendor scoring and selection

3. **Purchase Order Generation**
   - Automatic PO creation from selected RFQs
   - Integration with Odoo purchase module

4. **Goods Receipt Notes**
   - GRN creation and validation
   - Three-way matching (3WM) process

Vendor Management
-----------------

* Vendor catalogs with pricing and availability
* Vendor performance scoring
* Automated vendor selection based on criteria
