# IPAI Core

Core utilities and UX enhancements for InsightPulse AI ERP system.

## Features

- **IPAI Code**: Custom identification field for partners
- **Display Name Enhancement**: Automatically includes IPAI code in partner display names
- **Security Groups**: Dedicated user and manager groups for IPAI access control

## Installation

1. Update apps list: Settings → Apps → Update Apps List
2. Search for "IPAI Core"
3. Click Install

## Configuration

After installation:

1. Navigate to Contacts
2. Open any partner record
3. Set the "IPAI Code" field (e.g., IPAI-001)
4. The display name will automatically include the code

## Security

- **IPAI User**: Basic access to IPAI features
- **IPAI Manager**: Full access with create/delete permissions
