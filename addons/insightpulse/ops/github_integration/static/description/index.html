GitHub Integration Module

Integrate GitHub with Odoo via the pulser-hub GitHub App

✨ Features

📡 Webhook Endpoint
/odoo/github/webhook - Receive and process GitHub events
🔐 OAuth Integration
/odoo/github/auth/callback - Handle app installation authorization
📋 Track Pull Requests
Monitor PRs, approvals, and merge status in Odoo
🐛 Track Issues
Sync GitHub issues with Odoo project tasks
🔄 Bidirectional Sync
Create GitHub issues from Odoo tasks and vice versa
🤖 Bot Commands
/odoo-sync, /odoo-link, /odoo-status

📦 Installation

  1. Install Python dependencies: pip install PyJWT requests
  2. Install module in Odoo: Apps → Search "GitHub Integration" → Install
  3. Configure system parameters (see documentation)

⚙️ Configuration Required

⚠️ Important
You must configure these system parameters:
  • github.client_secret - GitHub App client secret
  • github.webhook_secret - Webhook secret for signature verification
  • github.private_key - Private key for API authentication
  • github.installation_id - Installation ID for your organization
Navigate to: Settings → Technical → Parameters → System Parameters
