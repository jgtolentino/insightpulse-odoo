ECharts Library Required
========================

Download echarts.min.js from:
https://cdn.jsdelivr.net/npm/echarts@5/dist/echarts.min.js

Save as: echarts.min.js

Or install via npm:
npm install echarts
cp node_modules/echarts/dist/echarts.min.js ./

The dashboard at /ipai/finance/ppm requires this library to render charts.
