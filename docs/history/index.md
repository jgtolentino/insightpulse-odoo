## Archive Policy

**When to Archive**:
- Documentation superseded by newer versions
- Features removed from product
- Technologies replaced
- Policies deprecated

**How to Archive**:
1. Move file to docs/history/
2. Add "ARCHIVED" prefix to filename
3. Update this index with archive date and reason
4. Remove from main navigation
